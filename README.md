# 💰 Python Personal Finance Tracker

A command-line interface (CLI) application built with **Python** to help students manage their daily expenses and track their budget efficiently.

## 🌟 Key Features
- **Expense Tracking:** Record daily spending with categories (Food, Transport, etc.).
- **Data Persistence:** Automatically saves all records into a `JSON` file.
- **Visual Summary:** Generates a summary showing total spending and percentage breakdown per category.
- **Error Handling:** Built-in validation for numeric inputs.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** `json`, `os`, `datetime` (No external installation required!)

## 🚀 How to Run
1. Ensure you have Python installed: `python --version`
2. Download the file: `finance_tracker.py`
3. Run the script: `python finance_tracker.py`

## 📂 Project Logic
This project demonstrates:
- **File I/O:** Reading and writing persistent data.
- **Data Structures:** Using Lists and Dictionaries for effective data management.
- **Modular Programming:** Organized class-based structure.
